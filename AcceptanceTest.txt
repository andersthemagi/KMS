Login menu:
  - Log in as customer -- displays the customer UI.
  - Log in as staff -- displays the staff UI.

Customer Select Item:
  - Item is added to the order list and total price is updated.

Customer Confirm order
  - confirmation box appears
      + customer confirms
          - message appears saying that order has been sent to staff
          - order appears on Staff UI
      + customer cancels
          - confirmation box disappears and customer can select more items

Customer log out:
  - if the customer logs back in, it is a fresh UI window (no previous order is displayed)

Staff Log in:
  - if the any customer has submitted an order, all orders will display on the screen
  - if no orders have been displayed, no text will appear in the order list.
